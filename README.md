# 🧠 100 Logic and Programming Exercises in Python

Este repositório contém 100 exercícios de lógica e programação resolvidos em Python.  
O objetivo é praticar raciocínio, estruturas básicas e resolução de problemas.  
Cada exercício está num ficheiro separado, simples e direto.

This repository contains 100 logic and programming exercises solved in Python.  
The goal is to practice reasoning, basic structures, and problem‑solving skills.  
Each exercise is stored in its own file, simple and easy to read.
