## 🔍 Motor de Pesquisa Inteligente | Intelligent Search Engine

Este repositório inclui uma aplicação de pesquisa avançada, capaz de encontrar exercícios por:

- palavras chave  
- frases completas  
- intenção (ação, objeto, objetivo)  
- sinónimos e variações automáticas  
- temas semânticos  
- ranking inteligente  

A aplicação permite:

- navegar pelos resultados com as setas ← →  
- destacar automaticamente os termos pesquisados  
- abrir exercícios diretamente no VS Code  
- pesquisar de forma natural, como:  
  - “remover duplicados de uma lista”  
  - “contar até 10”  
  - “separar texto por vírgulas”  
  - “ordenar números por ordem decrescente”  

### English
This repository also includes an advanced semantic search engine, capable of finding exercises by:

- keywords  
- full natural language phrases  
- intention (action, object, goal)  
- synonyms and automatic variations  
- semantic themes  
- intelligent ranking  

The application supports:

- navigation with ← →  
- automatic highlighting of search terms  
- opening exercises directly in VS Code  
- natural queries such as:  
  - “remove duplicates from a list”  
  - “count up to 10”  
  - “split text by commas”  
  - “sort numbers descending”
